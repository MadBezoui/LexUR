\section{Computational study}
\label{sec:exp}

\subsection{Design and protocol}
All experiments are implemented in the accompanying modular Python reproducibility package. The evaluation protocol tests selection \emph{quality} via a held-out analysis: each method selects one alternative using only its own logic, and the pick is scored against preference models that no method used during selection. 

For a held-out utility $u$ (higher is better) with in-sample best $x^\star_u$ and worst $x^-_u$, the held-out loss of a recommended alternative $x$ is
\begin{equation}
L_u(x)=\frac{u(x^\star_u)-u(x)}{u(x^\star_u)-u(x^-_u)}\in[0,1].
\label{eq:heldoutloss}
\end{equation}
We report the mean held-out loss and the tail held-out loss ($\mathrm{CVaR}_{10\%}$), emphasizing that robust methods exchange average performance to bound worst-case disappointment. Table~\ref{tab:baselines} specifies the compared baselines.

\subsection{Main Benchmark: The Broadened Protocol}
\label{sec:protocol}
To guard against confirmation bias, we ran a frozen audit protocol encompassing \protocolMethods{} methods, ten held-out preference families (linear, Chebyshev, augmented ASF, CES, Choquet, satisficing, sparse-linear, $L_p$, OWA, max-min), and eight front geometries. This generated $\protocolInstances{}$ paired instances. The protocol fixes acceptance gates tied to specific claims, reported verbatim in Table~\ref{tab:gates}. 

LexUR achieves a tail-loss rank of $4.67$. Within the Nemenyi critical difference, LexUR is statistically indistinguishable from the strongest robust scalarizations (MMR $4.57$, VIKOR $4.71$, ASF $4.77$). Single-point HV achieves the best average rank ($4.34$), though it returns a bare point without semantic explanation. LexUR provides an auditable regret certificate while maintaining strict tail robustness.

\subsection{Smart-grid dispatch case study}
We applied LexUR to a six-objective stochastic economic dispatch, balancing cost, $\text{CO}_2$, unreliability, ramping wear, non-renewable generation, and peak-to-average gap. LexUR accepts a negligible increase in mean loss for a 23\% reduction in worst-case loss compared to distance/averaging methods. Crucially, its certificate (Figure~\ref{fig:cert}) highlights the exact binding vulnerabilities—peak-shaving and non-renewable fraction—offering an explicit explanation rather than an opaque point recommendation.
